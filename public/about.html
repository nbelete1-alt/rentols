About RenTols

What is RenTols?
RenTols is a web app that helps college students and first-time renters compare rental prices across different cities in one place.
The Problem
Rental prices are scattered across many websites. RenTols puts it all in one place so users can compare options easily.
Who Is This For?
College students, first-time renters, and professionals looking to relocate.
How to Use
1. Go to the Home page and enter a city and state.
2. Click Search to see listings and a price chart.
3. Go to Listings to see results on a map.
4. Click Save City to save a city for later.
5. Visit the Saved page to see your saved cities.
APIs Used
RentCast API - provides rental listings and average prices.
OpenCage Geocoding API - converts city names into map coordinates.
Tech Stack
HTML, CSS, JavaScript, Node.js, Supabase, Chart.js
